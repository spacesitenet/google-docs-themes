# Privacy Policy

Google Docs Themes does not collect, transmit, sell, or share personal data.

## Data Stored

The extension stores the following settings in Chrome storage:

- Whether the extension is enabled
- Selected theme ID
- Favorite theme IDs
- Custom theme colors

These settings are used only to apply the user's selected Google Docs theme.

## Data Collection

Google Docs Themes does not:

- Collect browsing history
- Read document contents for analytics or storage
- Send data to external servers
- Use cookies
- Include analytics, ads, or tracking pixels

## Permissions

The extension requests:

- `storage`: saves theme preferences
- `https://docs.google.com/*`: applies themes on Google Docs pages

## Contact

Open a GitHub issue in this repository for privacy questions or concerns.
