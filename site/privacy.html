Google Docs Themes Privacy Policy

Google Docs Themes does not collect, transmit, sell, or share personal data. The extension is designed to run locally in Chrome and only stores theme preferences needed to provide its single purpose.

Data Stored

The extension stores these preferences in Chrome storage:

  • Whether the extension is enabled
  • The selected theme ID
  • Favorite theme IDs
  • Custom theme colors

These settings are used only to apply the user's selected Google Docs theme.

Data Collection

Google Docs Themes does not collect or transmit:

  • Personally identifiable information
  • Authentication information
  • Financial or health information
  • Location data
  • Browsing history
  • User activity such as clicks, keystrokes, mouse movement, or scrolling
  • Google Docs document content

Third Parties

Google Docs Themes does not sell user data, transfer user data to third parties, use analytics, show ads, or load remote code.

Permissions

  • storage: saves theme preferences.
  • https://docs.google.com/*: applies themes on Google Docs pages.

Contact

For privacy questions, open an issue on the GitHub repository.
